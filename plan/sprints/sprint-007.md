# Sprint 007: NPC 트레이너 / 체육관

## 목표
NPC 트레이너와 전투하고, 8개 체육관 수호자를 격파하여 인장을 획득한다.

## 대상 기능
- F-007: NPC 트레이너, 수호자, 체육관 인장

## 에셋 요구사항
- 체육관 관장 캐릭터 1종 (8방향, idle)
- 일반 트레이너 2종 (4방향, idle)
- 체육관 내부 타일셋 (나무 바닥 + 벽 패널)

## 선행 조건
Sprint 006 산출물 (대화 시스템, NPC 배치)

## 작업 항목
- [ ] PixelLab으로 트레이너/관장 캐릭터 + 체육관 타일셋 생성
- [ ] NPC 트레이너 데이터 로드 (maps.json trainers)
- [ ] 트레이너 전투 (도주 불가, 복수 몬스터 연전)
- [ ] 트레이너 AI (타입 상성 평가, 교체 판단)
- [ ] 격파 추적 (1회 승리 → 재도전 불가)
- [ ] 체육관 구조 (부하 2명 + 리더)
- [ ] 인장 획득 + 지역 해금 연동

## 완료 조건 (Done Criteria)
- [ ] 트레이너 접근 시 전투 시작 (검증: 전투 화면 전환)
- [ ] 트레이너전 도주 비활성화 (검증: UI 상태)
- [ ] 격파 트레이너 재도전 불가 (검증: 재접근 미발생)
- [ ] 관장 격파 → 인장 +1 (검증: 전후 비교)
- [ ] 인장 획득 → 새 지역 해금 (검증: 진입 가능 확인)

## 예상 파일 구조
```
src/world/npc.js            # (수정) 트레이너 팀, AI
src/core/battle.js          # (수정) 트레이너 전투 모드
src/world/map.js            # (수정) 인장 게이트
src/assets/sprites/npcs/    # 트레이너 에셋
src/assets/tilesets/        # 체육관 타일셋
```
