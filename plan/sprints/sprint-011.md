# Sprint 011: 오디오 / 난이도 / 배속

## 목표
칩튠 BGM과 효과음이 재생되고, 난이도 3단계와 전투 배속/오토 배틀이 설정 가능하다.

## 대상 기능
- F-011: 배속, 오토배틀
- F-012 일부: 오디오, 난이도

## 에셋 요구사항
- 없음 (이 스프린트는 오디오 + 설정 시스템 중심)

## 선행 조건
Sprint 010 산출물 (전체 게임 루프 완성)

## 작업 항목
- [ ] Web Audio API 칩튠 생성기
- [ ] 화면별 BGM (타이틀/맵/전투/메뉴/보스)
- [ ] 효과음 (공격/레벨업/포획/상태이상/메뉴)
- [ ] 음량 조절 / 음소거
- [ ] 난이도 3단계 (적 스탯 보정, AI 지능)
- [ ] 배속 1x/2x/4x
- [ ] 오토 배틀 (AI 자동 스킬 선택)
- [ ] 전투 UI에 배속/오토 토글

## 완료 조건 (Done Criteria)
- [ ] 맵 BGM 재생 (검증: AudioContext 상태)
- [ ] 전투 진입 시 BGM 변경 (검증: 전환 후 확인)
- [ ] 난이도 "어려움" → 적 스탯 상향 (검증: 데미지 비교)
