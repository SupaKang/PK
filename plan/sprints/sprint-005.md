# Sprint 005: 포획 / 레벨업 / 진화 / 파티

## 목표
마석으로 야생 몬스터를 포획하고, 전투 EXP로 레벨업/진화하며, 6마리 파티를 관리한다.

## 대상 기능
- F-006: 포획, EXP, 레벨업, 진화, 파티 관리

## 에셋 요구사항
- 포획 도구 아이콘 3종 (마석/고급 마석/마스터 마석 — 16x16)
- Sprint 003 몬스터 4종의 진화형 (64x64, 더 크고 디테일)

## 선행 조건
Sprint 004 산출물 (전투 확장, 아이템 사용 기반)

## 작업 항목
- [ ] PixelLab으로 포획 도구 아이콘 3종 생성
- [ ] PixelLab으로 진화형 몬스터 4종 생성 (64x64)
- [ ] 마석 포획 로직 (포획률 × HP비율 × 상태이상 보정)
- [ ] 포획 연출 (마석 투척 → 흔들림 → 결과)
- [ ] EXP 획득 → 레벨업 (스탯 상승 + learnset 스킬 습득)
- [ ] 진화 시스템 (레벨 조건 → 연출 → 종 변환 + 스프라이트 교체)
- [ ] 파티 관리 메뉴 (6슬롯, 순서 교체, 스탯 확인)
- [ ] 유대(Bond) 시스템 (전투 참여 시 증가, 크리티컬 보정)

## 완료 조건 (Done Criteria)
- [ ] 마석 사용 → 포획 성공/실패 결정 (검증: 반복 시도 통계)
- [ ] 포획 성공 시 파티에 추가 (검증: 파티 목록 확인)
- [ ] 전투 승리 → EXP → 레벨업 → 스탯 상승 (검증: 전후 비교)
- [ ] 진화 조건 충족 시 종 변경 + 스프라이트 교체 (검증: ID/이름/스프라이트 비교)
- [ ] 파티 메뉴 6마리 표시/교체 (검증: 메뉴 UI 확인)
