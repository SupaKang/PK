# Pocket Kingdom (포켓 킹덤) - 게임 기획 문서

> **장르**: 몬스터 수집 턴제 RPG
> **플랫폼**: 웹 브라우저 (PC / 모바일)
> **엔진**: 순수 JavaScript + Canvas 2D (자체 엔진)
> **타겟 해상도**: 800 x 600
> **버전**: v3.0.0

---

## 1. 게임 시놉시스

### 1.1 한 줄 요약

마왕에게 정복당한 세계에서, 마석 계약자가 된 주인공이 몬스터와 유대를 맺고 8명의 수호자를 찾아 인장을 모아 마왕성에 도전하는 몬스터 수집 RPG.

### 1.2 컨셉 키워드

| 키워드 | 설명 |
|--------|------|
| **마석 계약** | 야생 몬스터와 마법 돌을 통해 유대를 맺는 핵심 메카닉 |
| **수호자 순례** | 8개 지역의 수호자를 찾아 인장을 수집하는 메인 퀘스트 |
| **선택과 결말** | 플레이어의 카르마(선/악 선택)에 따라 3가지 엔딩 분기 |
| **클래스 정체성** | 5개 직업 중 선택하여 주인공이 직접 전투에 참여 |

### 1.3 차별점

- 주인공(계약자)이 직접 전투에 참여하는 클래스 시스템
- 카르마 기반 멀티 엔딩 (빛 / 어둠 / 균형)
- 프로시저럴 픽셀아트 몬스터 스프라이트 생성
- 웹 브라우저에서 즉시 플레이 가능 (설치 불필요)

## 2. 스토리

### 2.1 세계관

먼 옛날, 인간과 몬스터는 **마석(Magic Stone)**을 매개로 계약을 맺고 공존하며 살았다. 그러나 어둠의 틈에서 **마족**이 출현하고, 그들의 왕 **마왕 카오스**가 이끄는 군세가 세계를 침략했다. **대전쟁**이 벌어졌지만 인간은 패배했고, 마족이 세계의 지배자가 되었다.

그러나 하나의 예언이 남아 있다:

> *"어둠이 세상을 삼킨 뒤, 잊혀진 마석의 빛이 다시 깨어나리라. 그 빛을 이끄는 자가 세계를 되찾으리."*

8명의 **수호자**가 각 지역에서 결계를 유지하며 인간의 마지막 희망을 지키고 있다.

### 2.2 주요 등장인물

| 캐릭터 | 역할 | 설명 |
|---------|------|------|
| **주인공 (계약자)** | 플레이어 캐릭터 | 마석 계약의 힘을 가진 예언의 아이. 5개 클래스 중 택 1 |
| **할아버지** | 멘토 / 내레이터 | 프롤로그에서 세계관을 전하는 인물. 이미 고인이며 꿈에서 등장 |
| **은하** | 라이벌 | 같은 날 여행을 시작한 밝고 경쟁적인 동료. 4회 대결 |
| **수호자 8인** | 체육관 리더 | 불꽃, 석진, 미나, 라이, 무현, 설화, 용왕, 성운 |
| **그림자단** | 적대 조직 | 마왕의 뜻을 따르는 인간 조직. 간부: 흑사, 검은달, 보스: 흑염 |
| **마왕 카오스** | 최종 보스 | 마족의 왕. 세계를 정복한 어둠의 존재 |
| **사천왕** | 엘리트 4인 | 마왕의 최강 부하 4인. 연속 배틀로 돌파 |

### 2.3 챕터 구성 (총 10챕터)

| 챕터 | 제목 | 주요 내용 |
|-------|------|----------|
| **0** | 오래된 이야기 | 프롤로그. 할아버지의 이야기 → 꿈에서 깨어남 → 클래스 선택 → 스타터 몬스터 선택 |
| **1** | 화염의 시련 | 시작의 마을 도착 → 라이벌 은하 첫 대결 → 화염촌 → 수호자 불꽃 (화염 인장) |
| **2** | 대지의 시련 | 바위동굴 → 그림자단 조우 → 수호자 석진 (대지 인장) |
| **3** | 해류의 시련 | 해류항 → 라이벌 2차 대결 → 수호자 미나 (해류 인장) |
| **4** | 전격의 시련 | 그림자단 간부 흑사 조우 → 수호자 라이 (전격 인장) |
| **5** | 투지의 시련 | 수호자 무현 (투지 인장) → 라이벌 3차 대결 |
| **6** | 빙결과 용의 시련 | 수호자 설화 (빙결 인장) → 간부 검은달 전투 → 수호자 용왕 (용린 인장) |
| **7** | 그림자의 습격 | 그림자단 비밀기지 → 보스 흑염 격파 → 마왕성 위치 확인 |
| **8** | 마지막 인장 | 수호자 성운 (우주 인장, 8번째) → 라이벌 최종 대결 → 사천왕 연전 |
| **9** | 새벽이 밝아온다 | 마왕 카오스 격파 → 카르마에 따른 엔딩 분기 → 후일담 |

### 2.4 엔딩 분기

| 엔딩 | 조건 | 결말 |
|------|------|------|
| **빛의 결말** | 카르마 +30 이상 | 마왕 격파 후 세계에 빛이 돌아옴. 인간, 몬스터, 마족이 공존하는 새 시대 |
| **어둠의 결말** | 카르마 -30 이하 | 마왕을 쓰러뜨렸으나, 어둠의 힘이 계약자에게 스며들어 새로운 마왕이 탄생 |
| **균형의 결말** | 그 외 | 어둠은 걷혔으나 완전히 사라지지 않음. 계약자의 여정은 계속됨 |

### 2.5 프롤로그 상세

게임은 깊은 밤, 할아버지가 화롯불 앞에서 낡은 책을 읽어주는 장면으로 시작된다. 대전쟁과 예언, 수호자의 이야기를 들은 후, 플레이어는 **클래스를 선택**하고 **스타터 몬스터**(불꽃냥/물방울이/새싹이 중 1)를 받는다.

그러나 장면이 전환되면 — 이것은 꿈이었다. 주인공은 숲 속에서 눈을 뜬다. 할아버지는 이미 오래전 돌아가신 분. 옆에는 어린 시절부터 함께한 파트너 **고랭이**가 잠들어 있다. 가방에 마석 몇 개와 물약을 챙기고, 저 너머에 보이는 마을을 향해 여정을 시작한다.

## 3. 게임 시스템

### 3.1 직업(클래스) 시스템

게임 시작 시 5개 직업 중 하나를 선택한다. 계약자(주인공)는 몬스터와 함께 직접 전투에 참여한다.

| 직업 | 타입 | HP | ATK | DEF | SpATK | SpDEF | SPD | 특징 |
|------|------|-----|-----|-----|-------|-------|-----|------|
| **전사** | 격투 | 75 | 85 | 70 | 30 | 45 | 55 | 물리 공격 특화. 궁극기: 전사의 일격 |
| **마법사** | 우주 | 50 | 30 | 40 | 90 | 70 | 60 | 원소 마법 다수. 궁극기: 코즈믹 레이 |
| **궁수** | 바람 | 55 | 75 | 45 | 60 | 50 | 85 | 최고 속도. 궁극기: 스카이 다이브 |
| **성직자** | 빛 | 70 | 35 | 55 | 75 | 85 | 40 | 치유/방어 특화. 궁극기: 디바인 라이트 |
| **암흑기사** | 어둠 | 65 | 80 | 60 | 65 | 55 | 50 | 공방 균형형, 느린 성장. 궁극기: 어비스 게이즈 |

- 각 직업은 Lv.1~50까지 고유 스킬 트리 보유
- Lv.40에 궁극기(Ultimate) 습득
- 계약자는 진화하지 않으며, 레벨업을 통해 성장

### 3.2 몬스터 시스템

#### 3.2.1 기본 구조
- **총 102종**의 몬스터
- 각 몬스터는 6개 기본 스탯 보유: HP, ATK, DEF, SpATK, SpDEF, SPD
- **개체값(IV)**: 스탯당 0~31 랜덤 → 같은 종이라도 개체 차이 존재
- **최대 레벨**: 100

#### 3.2.2 계약(포획) 시스템
- **마석(Magic Stone)**을 사용하여 야생 몬스터와 계약
- 성공률 = 몬스터별 포획률 × 현재 HP 비율 × 상태이상 보정
- 스타터(포획률 45) < 일반(높음) < 전설(매우 낮음)

#### 3.2.3 파티 구성
- 최대 **6마리** + 계약자 1인
- 전투 중 교체 가능
- 모든 파티 몬스터가 전투불능 시 패배

#### 3.2.4 진화 시스템
- 레벨 조건 충족 시 진화
- 분기 진화 체인 존재 (예: 불꽃냥 → 염수사자 → 염제왕)
- 진화 시 스탯 대폭 상승 + 새 스킬 습득

#### 3.2.5 유대(Bond) 시스템
- 몬스터별 유대도 0~255
- 높은 유대도 → 크리티컬 확률 증가, 데미지 보너스
- 전투 참여, 승리 시 유대도 상승

#### 3.2.6 스타터 몬스터

| 몬스터 | 타입 | 특성 | 설명 |
|--------|------|------|------|
| **고랭** | 노말 | 랜덤 특성 | 빠른 성장. 최종 진화: 나시고랭 |

### 3.3 전투 시스템

#### 3.3.1 기본 구조
- **턴제 1 vs 1** 형식
- 속도(SPD) 스탯이 턴 순서 결정 (우선도 스킬로 변경 가능)
- 행동 선택: **공격 / 교체 / 아이템 / 도주** (트레이너전에서는 도주 불가)

#### 3.3.2 스킬
- **총 346개** 스킬
- 3개 카테고리: **물리 / 특수 / 변화**
- 스킬별 속성: 위력, 명중률, PP(사용 횟수)
- 11가지 효과 유형: 다단 히트, 반동, 흡수, 치유, 상태이상, 능력치 변화, 보호, 우선도, 높은 크리티컬, 위력 스케일링(HP 비례/반전)

#### 3.3.3 타입 상성
- **18개 타입** 체계:

| | | | | | |
|---|---|---|---|---|---|
| 화염 | 해류 | 자연 | 전격 | 대지 | 빙결 |
| 바람 | 독 | 금속 | 빛 | 어둠 | 영혼 |
| 용 | 격투 | 바위 | 소리 | 우주 | 일반 |

- 상성 배율: 2배(효과적) / 0.5배(비효과적) / 0배(무효)
- 이중 타입 몬스터는 배율 중첩 (최대 4배)

#### 3.3.4 상태이상

| 상태이상 | 효과 |
|----------|------|
| **화상** | 매 턴 HP 감소, 물리 공격력 하락 |
| **독** | 매 턴 HP 감소, 특수 공격력 하락 |
| **마비** | 속도 하락, 25% 확률로 행동 불가 |
| **수면** | 행동 불가 (1~3턴 후 해제, 해제 후 수면 시간 비례 총체력의 10% 체력 회복) |
| **혼란** | 50% 확률로 자해 1~3턴 랜덤 |
| **빙결** | 행동 불가 (매 턴 50% 해제 확률), 불속성 빙결 불가, 물속성 빙결 해제확률 15%저하 |

#### 3.3.5 크리티컬 히트
- 기본 확률: 6.25%
- 유대도 높을수록 확률 증가
- 크리티컬 시 1.5배 데미지

#### 3.3.6 AI 시스템
- **트레이너 AI**: 타입 상성, HP 비율, 상태이상을 종합 평가하여 최적 행동 선택. 불리한 상성 시 교체 판단
- **야생 몬스터**: 랜덤 행동 (위력 가중치)
- **난이도 3단계**: AI 지능 및 적 스탯 보정

### 3.7 그림자단 & 보스 전투

#### 3.7.1 그림자단 조우

| 시점 | 적 | 팀 레벨 | 보상 |
|------|-----|---------|------|
| 챕터 2 | 그림자단 졸개 | Lv.10~12 | 500G |
| 챕터 4 | 간부 흑사 | Lv.22~24 | 2,000G |
| 챕터 6 | 간부 검은달 | Lv.38~42 | 4,000G |
| 챕터 7 | 보스 흑염 | Lv.44~48 (5마리) | 8,000G |

#### 3.7.2 최종전: 사천왕 + 마왕
- 인장 8개 수집 후 마왕성 진입
- **사천왕 4연전** (중간 회복 없음)
- **마왕 카오스** 최종 배틀
- 승리 시 카르마에 따른 엔딩 분기

### 3.8 아이템 시스템

#### 3.8.1 아이템 분류

| 카테고리 | 예시 | 용도 |
|----------|------|------|
| **회복 아이템** | 물약, 상처약, 완전회복 | HP/상태이상 치료 |
| **계약 아이템** | 마석, 고급 마석 | 야생 몬스터 포획 |
| **전투 보조** | 해독제, 화상약, 각성제 | 특정 상태이상 해제 |
| **탐험 도구** | 탈출 로프, 식량 | 탐험 보조 |

- **42종** 아이템
- 마을 상점에서 구입 가능 (지역별 상이한 라인업)
- 시작 지급: 물약 5개 + 마석 10개 + 3,000G

### 3.9 세이브 & 진행

- **3개 세이브 슬롯** (LocalStorage 기반)
- 마을 진입 시 자동 저장
- 메뉴에서 수동 저장/불러오기 가능

### 3.10 도감 시스템

- 102종 몬스터 도감
- **목격 / 계약** 별도 카운트
- 서식지 정보, 스탯, 설명 열람

### 3.11 업적 시스템

총 **15개** 업적 마일스톤:

- 첫 몬스터 계약
- 레벨 마일스톤 달성 (10, 25, 50, 100)
- 체육관 클리어 (1~8번째)
- 사천왕 돌파
- 마왕 격파
- 도감 완성 등

### 3.12 난이도 시스템

| 난이도 | 적 스탯 보정 | AI 지능 |
|--------|-------------|---------|
| **쉬움** | 하향 | 단순 |
| **보통** | 기본 | 표준 |
| **어려움** | 상향 | 고급 (최적 상성 교체, 상태이상 활용) |

### 3.13 후반 콘텐츠

- 인장 8개 수집 후 **전설 몬스터 8종** 출현 해금
- 각 전설 몬스터는 특정 지역에 초저확률 등장 (Lv.50~55)
- 도감 완성 도전
- 뉴게임+: 강화된 적과 재도전

## 4. UI & 화면 구성

### 4.1 화면 흐름도

```
[타이틀] → [새 게임 / 불러오기 / 옵션]
              ↓
         [프롤로그 / 클래스 선택 / 스타터 선택]
              ↓
         [맵 화면] ←→ [전투 화면]
              ↓↑         ↓↑
         [메뉴]       [전투 종료]
          ├─ 파티
          ├─ 가방
          ├─ 도감
          ├─ 업적
          ├─ 저장/불러오기
          └─ 설정
              ↓
         [상점 화면]
              ↓
         [대화 / 선택지 화면]
```

### 4.2 주요 화면

| 화면 | 설명 |
|------|------|
| **타이틀** | 별이 흐르는 애니메이션 배경. 새 게임/불러오기/옵션 |
| **맵** | 타일맵 기반. 플레이어 스프라이트, NPC, 미니맵 토글 |
| **전투** | 적/아군 몬스터 스프라이트 대치. 플로팅 데미지, 상성 표시, 상태이상 아이콘 |
| **메뉴** | 파티(6슬롯), 가방(카테고리별), 도감, 스탯, 저장 |
| **대화** | 타자기 효과 텍스트 + 화자명 + 분기 선택지 |
| **상점** | 아이템 목록 + 가격 + 구입/판매 |

### 4.3 전투 UI 상세

- 배속 조절: 1x / 2x / 4x
- 오토 배틀 옵션
- 배틀 로그 실시간 표시
- 타입 상성 효과 메시지 ("효과가 굉장했다!" / "효과가 별로인 듯하다...")
- 플로팅 데미지 숫자 애니메이션

## 6. 핵심 게임 루프

```
[마을]
  ├─ 상점에서 아이템 구매
  ├─ 파티 편성/관리
  └─ 저장
       ↓
[루트/동굴 탐험]
  ├─ 야생 몬스터 조우 → 전투 or 도주
  │   └─ 승리 → EXP 획득 → 레벨업/진화
  │   └─ 계약 시도 → 새 몬스터 획득
  ├─ NPC 트레이너 전투 → 보상 획득
  └─ 스토리 이벤트 발생
       ↓
[체육관]
  ├─ 부하 트레이너 격파
  └─ 수호자 격파 → 인장 획득 → 새 지역 해금
       ↓
[다음 지역으로 진행]
       ↓ (반복)
[마왕성] → 사천왕 연전 → 마왕 격파 → 엔딩
```
