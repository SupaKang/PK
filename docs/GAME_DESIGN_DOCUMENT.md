# Pocket Kingdom (포켓 킹덤) - 게임 기획 문서

> **장르**: 크리처 수집 턴제 RPG
> **플랫폼**: 웹 브라우저 (PC / 모바일)
> **엔진**: 순수 JavaScript + Canvas 2D (자체 엔진)
> **타겟 해상도**: 800 x 600
> **버전**: v4.0.0

---

## 1. 게임 시놉시스

### 1.1 한 줄 요약

마왕에게 정복당한 세계에서, 마석 계약자가 된 주인공이 크리처와 유대를 맺고 8명의 수호자를 찾아 인장을 모아 카오스 성채에 도전하는 크리처 수집 RPG.

### 1.2 컨셉 키워드

| 키워드            | 설명                                                     |
| ----------------- | -------------------------------------------------------- |
| **마석 계약**     | 야생 크리처와 마법 돌을 통해 유대를 맺는 핵심 메카닉     |
| **수호자 순례**   | 8개 지역의 수호자를 찾아 인장을 수집하는 메인 퀘스트     |
| **선택과 결말**   | 플레이어의 카르마(선/악 선택)에 따라 3가지 엔딩 분기     |
| **클래스 정체성** | 5개 직업 중 선택하여 주인공이 직접 전투에 참여           |

### 1.3 차별점

- 주인공(계약자)이 직접 전투에 참여하는 클래스 시스템
- 카르마 기반 멀티 엔딩 (빛 / 어둠 / 균형)
- 프로시저럴 픽셀아트 크리처 스프라이트 생성
- 웹 브라우저에서 즉시 플레이 가능 (설치 불필요)

## 2. 스토리

### 2.1 세계관

먼 옛날, 인간과 크리처는 **마석(Magic Stone)**을 매개로 교감하고 공존하며 살았다. 그러나 어둠의 틈에서 **마족**이 출현하고, 그들의 왕 **마왕 카오스**가 이끄는 군세가 세계를 침략했다. **대전쟁**이 벌어졌지만 인간과 크리처는 패배했고, 마족이 세계의 지배자가 되었다.

그러나 하나의 예언이 남아 있다:

> _"어둠이 세상을 삼킨 뒤, 잊혀진 영웅이 다시 깨어나리라. 그가 이끄는 빛이 세계에 평화를 되찾으리라."_

8명의 **수호자**가 각 지역에서 결계를 유지하며 인간의 마지막 희망을 지키고 있다.

### 2.2 주요 등장인물

| 캐릭터              | 역할           | 설명                                                         |
| ------------------- | -------------- | ------------------------------------------------------------ |
| **주인공 (계약자)** | 플레이어 캐릭터 | 마석 계약의 힘을 가진 예언의 아이. 5개 클래스 중 택 1        |
| **할아버지**        | 멘토 / 내레이터 | 프롤로그에서 세계관을 전하는 인물. 이미 고인이며 꿈에서 등장 |
| **은하**            | 라이벌          | 같은 날 여행을 시작한 밝고 경쟁적인 동료. 4회 대결           |
| **수호자 8인**      | 전당 수호자     | 불꽃, 석진, 미나, 라이, —, 무현, 설화, 용왕, 성운            |
| **그림자단**        | 적대 조직       | 마왕의 뜻을 따르는 인간 조직. 간부: 흑사, 검은달, 보스: 흑염 |
| **마왕 카오스**     | 최종 보스       | 마족의 왕. 세계를 정복한 어둠의 존재                         |
| **사천왕**          | 엘리트 4인      | 마왕의 최강 부하 4인. 연속 배틀로 돌파                       |

| 챕터  | 제목             | 주요 내용                                                                   |
| ----- | ---------------- | --------------------------------------------------------------------------- |

| **1** | 화염의 시련      | 아스텔 도착 → 라이벌 은하 첫 대결 → 칼데라 → 수호자 불꽃 (아그니온, 화염 인장) |

| **3** | 해류의 시련      | 포르테나 → 라이벌 2차 대결 → 수호자 미나 (네레이드, 해류 인장)              |
| **4** | 전격의 시련      | 그림자단 간부 흑사 조우 → 수호자 라이 (케라노스, 전격 인장)                 |

| **6** | 빙결과 용의 시련 | 수호자 설화 (아우로라, 빙결 인장) → 간부 검은달 → 수호자 용왕 (드라코니아, 용린 인장) |

| **8** | 마지막 인장      | 수호자 성운 (아스트라, 우주 인장) → 라이벌 최종 대결 → 사천왕 연전          |
| **9** | 새벽이 밝아온다  | 마왕 카오스 격파 → 카르마에 따른 엔딩 분기 → 후일담                         |

| **어둠의 결말** | 카르마 -30 이하 | 마왕을 쓰러뜨렸으나, 어둠의 힘이 계약자에게 스며들어 새로운 마왕이 탄생   |
| **균형의 결말** | 그 외           | 어둠은 걷혔으나 완전히 사라지지 않음. 계약자의 여정은 계속됨              |

### 2.5 프롤로그 상세

게임은 깊은 밤, 할아버지가 화롯불 앞에서 낡은 책을 읽어주는 장면으로 시작된다. 대전쟁과 예언, 수호자의 이야기를 들은 후, 플레이어는 **클래스를 선택**한다.

그러나 장면이 전환되면 — 이것은 꿈이었다. 주인공은 숲 속에서 눈을 뜬다. 할아버지는 이미 오래전 돌아가신 분. 옆에는 어린 시절부터 함께한 파트너 **스트레이**가 잠들어 있다. 가방에 마석 몇 개와 물약을 챙기고, 저 너머에 보이는 마을을 향해 여정을 시작한다.

## 3. 게임 시스템

### 3.1 직업(클래스) 시스템

게임 시작 시 5개 직업 중 하나를 선택한다. 계약자(주인공)는 크리처와 함께 직접 전투에 참여한다.

| 직업         | 타입 | HP  | ATK | DEF | SpATK | SpDEF | SPD | 특징                                          |
| ------------ | ---- | --- | --- | --- | ----- | ----- | --- | --------------------------------------------- |
| **전사**     | 격투 | 75  | 85  | 70  | 30    | 45    | 55  | 물리 공격 특화. 궁극기: 전사의 일격           |
| **마법사**   | 우주 | 50  | 30  | 40  | 90    | 70    | 60  | 원소 마법 다수. 궁극기: 코즈믹 레이           |
| **궁수**     | 바람 | 55  | 75  | 45  | 60    | 50    | 85  | 최고 속도. 궁극기: 스카이 다이브              |
| **성직자**   | 빛   | 70  | 35  | 55  | 75    | 85    | 40  | 치유/방어 특화. 궁극기: 디바인 라이트         |
| **암흑기사** | 어둠 | 65  | 80  | 60  | 65    | 55    | 50  | 공방 균형형, 느린 성장. 궁극기: 어비스 게이즈 |

- 각 직업은 Lv.1~50까지 고유 스킬 트리 보유
- Lv.40에 궁극기(Ultimate) 습득
- 계약자는 진화하지 않으며, 레벨업과 장비를 통해 성장

### 3.2 장비 시스템

계약자(주인공)만 장비를 착용할 수 있다. 총 **5슬롯**: 무기 1 + 방어구 3 + 장신구 1.

#### 3.2.1 장비 슬롯

| 슬롯       | 수량 | 주요 효과              | 예시                         |
| ---------- | ---- | ---------------------- | ---------------------------- |
| **무기**   | 1    | ATK, SpATK, 크리티컬   | 검, 지팡이, 활, 철퇴, 단검  |
| **투구**   | 1    | HP, SpDEF              | 가죽모자, 철투구, 마법관     |
| **갑옷**   | 1    | DEF, HP                | 가죽갑옷, 사슬갑옷, 로브    |
| **장갑**   | 1    | ATK, DEF, 명중률       | 가죽장갑, 철건틀릿, 마법장갑 |
| **장신구** | 1    | 특수 효과, 복합 스탯   | 반지, 목걸이, 부적, 귀걸이   |

#### 3.2.2 장비 등급

| 등급       | 색상   | 스탯 보정 | 획득 방법                |
| ---------- | ------ | --------- | ------------------------ |
| **일반**   | 흰색   | 기본      | 상점 구매, 필드 드롭     |
| **고급**   | 녹색   | +15%      | 상점 (후반), 보스 드롭   |
| **희귀**   | 파란색 | +30%      | 던전 보상, 수호자 보상   |
| **전설**   | 보라색 | +50% + 특수 효과 | 히든 이벤트, 최종 보스 |

#### 3.2.3 직업별 장비 제한

| 직업     | 무기            | 갑옷           | 특징                     |
| -------- | --------------- | -------------- | ------------------------ |
| 전사     | 검, 철퇴        | 중갑           | 고 DEF, 고 ATK           |
| 마법사   | 지팡이          | 로브           | 고 SpATK, 고 SpDEF       |
| 궁수     | 활              | 경갑           | 고 SPD, 고 명중          |
| 성직자   | 지팡이, 철퇴    | 중갑, 로브     | 회복 보너스, 고 SpDEF    |
| 암흑기사 | 검, 단검        | 중갑, 경갑     | 흡수 효과, 균형 스탯     |

#### 3.2.4 장비 획득

- **상점**: 마을마다 해당 권역에 맞는 장비 판매 (인장 진행도에 따라 라인업 확장)
- **드롭**: 트레이너/보스 격파 시 확률 드롭
- **수호자 보상**: 수호자 격파 시 해당 타입의 희귀 장비 1개 확정 지급
- **히든 이벤트**: 던전 숨겨진 상자에서 전설 장비 획득
- **제작**: 없음 (드롭/구매 기반)

#### 3.2.5 장신구 특수 효과 예시

| 장신구           | 등급 | 효과                                |
| ---------------- | ---- | ----------------------------------- |
| 생명의 반지      | 고급 | 매 턴 HP 5% 자동 회복              |
| 속도의 귀걸이    | 고급 | SPD +15                            |
| 수호자의 목걸이  | 희귀 | 전 스탯 +5                         |
| 인장의 부적      | 희귀 | 인장 수 × ATK +2 보너스            |
| 카오스의 반지    | 전설 | ATK/SpATK +20, DEF/SpDEF -10       |
| 여명의 펜던트    | 전설 | HP 20% 이하 시 전 스탯 1.5배       |

### 3.3 크리처 시스템

#### 3.3.1 기본 구조

- **총 102종**의 크리처
- 각 크리처는 6개 기본 스탯 보유: HP, ATK, DEF, SpATK, SpDEF, SPD
- **개체값(IV)**: 스탯당 0~31 랜덤 → 같은 종이라도 개체 차이 존재
- **최대 레벨**: 100

#### 3.3.2 계약(포획) 시스템

- **마석(Magic Stone)**을 사용하여 야생 크리처와 계약
- 성공률 = 크리처별 포획률 × 현재 HP 비율 × 상태이상 보정
- 스타터(포획률 45) < 일반(높음) < 전설(매우 낮음)

#### 3.3.3 파티 구성

- 최대 **6마리** + 계약자 1인
- 전투 중 교체 가능
- 모든 파티 크리처가 전투불능 시 패배

#### 3.3.4 진화 시스템

- 레벨 조건 충족 시 진화
- 분기 진화 체인 존재 (예: 신더 → 이그니온 → 이프리트)
- 진화 시 스탯 대폭 상승 + 새 스킬 습득

#### 3.3.5 유대(Bond) 시스템

- 크리처별 유대도 0~255
- 높은 유대도 → 크리티컬 확률 증가, 데미지 보너스
- 전투 참여, 승리 시 유대도 상승

#### 3.3.6 동반 크리처

| 크리처       | 타입 | 특성      | 설명                                       |
| ------------ | ---- | --------- | ------------------------------------------ |
| **스트레이** | 노말 | 랜덤 특성 | 빠른 성장. 최종 진화: 나이트 스토커         |

### 3.4 전투 시스템

#### 3.4.1 기본 구조

- **턴제 1 vs 1** 형식
- 속도(SPD) 스탯이 턴 순서 결정 (우선도 스킬로 변경 가능)
- 행동 선택: **공격 / 교체 / 아이템 / 도주** (트레이너전에서는 도주 불가)

#### 3.4.2 스킬

- **총 346개** 스킬
- 3개 카테고리: **물리 / 특수 / 변화**
- 스킬별 속성: 위력, 명중률, PP(사용 횟수)
- 11가지 효과 유형: 다단 히트, 반동, 흡수, 치유, 상태이상, 능력치 변화, 보호, 우선도, 높은 크리티컬, 위력 스케일링(HP 비례/반전)

#### 3.4.3 타입 상성

- **18개 타입** 체계:

|      |      |      |      |      |      |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 화염 | 해류 | 자연 | 전격 | 대지 | 빙결 |
| 바람 | 독   | 금속 | 빛   | 어둠 | 영혼 |
| 용   | 격투 | 바위 | 소리 | 우주 | 일반 |

- 상성 배율: 2배(효과적) / 0.5배(비효과적) / 0배(무효)
- 이중 타입 크리처는 배율 중첩 (최대 4배)

#### 3.4.4 상태이상

| 상태이상 | 효과                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| **화상** | 매 턴 HP 감소, 물리 공격력 하락                                                 |
| **독**   | 매 턴 HP 감소, 특수 공격력 하락                                                 |
| **마비** | 속도 하락, 25% 확률로 행동 불가                                                 |
| **수면** | 행동 불가 (1~3턴 후 해제, 해제 후 수면 시간 비례 총체력의 10% 체력 회복)        |
| **혼란** | 50% 확률로 자해 1~3턴 랜덤                                                      |
| **빙결** | 행동 불가 (매 턴 50% 해제 확률), 불속성 빙결 불가, 물속성 빙결 해제확률 15%저하 |

#### 3.4.5 크리티컬 히트

- 기본 확률: 6.25%
- 유대도 높을수록 확률 증가
- 크리티컬 시 1.5배 데미지

#### 3.4.6 AI 시스템

- **트레이너 AI**: 타입 상성, HP 비율, 상태이상을 종합 평가하여 최적 행동 선택. 불리한 상성 시 교체 판단
- **야생 크리처**: 랜덤 행동 (위력 가중치)
- **난이도 3단계**: AI 지능 및 적 스탯 보정

### 3.5 월드 & 탐험

### 3.14 퀘스트 시스템

퀘스트는 **메인 퀘스트**와 **사이드 퀘스트**로 나뉜다. 메인 퀘스트가 스토리 진행의 뼈대이며, 사이드 퀘스트는 탐험 보상과 세계관 몰입을 강화한다.

#### 퀘스트 UI

- 퀘스트 로그: 메뉴에서 현재/완료 퀘스트 목록 확인

- 퀘스트 마커: 맵에서 목표 NPC/위치에 ! 또는 ? 아이콘
- 메인 퀘스트는 금색, 사이드 퀘스트는 은색으로 구분

#### 3.14.2 사이드 퀘스트

각 마을에 사이드 퀘스트를 배치하여 탐험 보상과 세계관 몰입을 강화한다. 메인 퀘스트와 독립적으로 진행 가능.

#### 퀘스트 구조

- NPC 대화로 수주 → 조건 달성 → NPC에게 보고 → 보상
- 퀘스트 로그에서 진행 상황 확인 가능
- 일부 퀘스트는 인장 수 또는 선행 퀘스트 필요

#### 아스텔 (시작 마을)

| 퀘스트명 | 의뢰인 | 내용 | 보상 |
|----------|--------|------|------|
| 잃어버린 고양이 | 마을 소녀 | 마을 외곽에서 도망친 크리처를 찾아 데려오기 | 포션 ×3, 300G |
| 할아버지의 유품 | 마을 촌장 | 카른 평원에서 낡은 책 회수 | 마석 ×5, 마을 지도 |
| 첫 번째 계약 | 여행자 | 야생 크리처 1마리 계약 후 보고 | 고급 마석 ×2 |

## 4. UI & 화면 구성

### 4.1 화면 흐름도

```
[타이틀] → [새 게임 / 불러오기 / 옵션]
              ↓
         [프롤로그 / 클래스 선택]
              ↓
         [맵 화면] ←→ [전투 화면]
              ↓↑         ↓↑
         [메뉴]       [전투 종료]
          ├─ 파티
          ├─ 가방
          ├─ 도감
          ├─ 업적
          ├─ 저장/불러오기
          └─ 설정
              ↓
         [상점 화면]
              ↓
         [대화 / 선택지 화면]
```

### 4.2 주요 화면

| 화면       | 설명                                                                       |
| ---------- | -------------------------------------------------------------------------- |
| **타이틀** | 별이 흐르는 애니메이션 배경. 새 게임/불러오기/옵션                         |
| **맵**     | 타일맵 기반. 플레이어 스프라이트, NPC, 미니맵 토글                         |
| **전투**   | 적/아군 크리처 스프라이트 대치. 플로팅 데미지, 상성 표시, 상태이상 아이콘  |
| **메뉴**   | 파티(6슬롯), 가방(카테고리별), 도감, 스탯, 저장                            |
| **대화**   | 타자기 효과 텍스트 + 화자명 + 분기 선택지                                  |
| **상점**   | 아이템 목록 + 가격 + 구입/판매                                             |

### 4.3 전투 UI 상세

- 배속 조절: 1x / 2x / 4x
- 오토 배틀 옵션
- 배틀 로그 실시간 표시
- 타입 상성 효과 메시지 ("효과가 굉장했다!" / "효과가 별로인 듯하다...")
- 플로팅 데미지 숫자 애니메이션

## 6. 핵심 게임 루프

```
[마을]
  ├─ 상점에서 아이템 구매
  ├─ 파티 편성/관리
  └─ 저장
       ↓
[루트/동굴 탐험]
  ├─ 야생 크리처 조우 → 전투 or 도주
  │   └─ 승리 → EXP 획득 → 레벨업/진화
  │   └─ 계약 시도 → 새 크리처 획득
  ├─ NPC 트레이너 전투 → 보상 획득
  └─ 스토리 이벤트 발생
       ↓
[수호자 전당]
  ├─ 부하 트레이너 격파
  └─ 수호자 격파 → 인장 획득 → 새 지역 해금
       ↓
[다음 지역으로 진행]
       ↓ (반복)
[카오스 성채] → 사천왕 연전 → 마왕 격파 → 엔딩
```
