# Pocket Kingdom (포켓 킹덤)

마석 계약자의 타임어택 탐험 RPG

## 게임 소개

마족이 지배하는 세계에서, 계약자가 되어 8명의 수호자를 만나고 마왕을 쓰러뜨리는 이야기.

### 핵심 시스템
- **타임어택 탐험**: 240AP 제한, 시간 내 목표 달성 + 귀환
- **시간대 시스템**: 아침/낮/저녁/밤 — 밤에는 강한 마물 출현
- **캠핑**: 시간을 써서 회복 vs 위험을 감수하고 전진
- **5가지 직업**: 전사/마법사/궁수/성직자/암흑기사
- **마석 계약**: 야생 몬스터와 계약하여 파트너로

### 조작법
- **방향키**: 이동
- **Enter/Space**: 상호작용
- **Escape**: 메뉴
- **C**: 캠핑
- **M**: 미니맵
- **H**: 도움말
- **I**: 전투 중 적 정보
- **S**: 전투 속도 조절

## 개발 환경

```bash
npm install
npm run dev     # 개발 서버 (localhost:8080)
npm run build   # 프로덕션 빌드
```
