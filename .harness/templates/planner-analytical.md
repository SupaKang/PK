## 분석/시각화 도메인 분해 규칙

기능 분해 시 다음 계층을 따르세요:

1. **Data Acquisition**: 데이터 획득
   - "API에서 관측 데이터를 조회할 수 있다"
   - "CSV/Excel 파일에서 데이터를 읽어올 수 있다"

2. **Preprocessing**: 전처리
   - "결측치를 적절한 방법으로 보간한다"
   - "단위를 통일한다 (예: mm→m, hPa→kPa)"
   - "시간대를 표준화한다"

3. **Analysis**: 분석
   - "기간별 통계(평균, 최대, 최소)를 산출한다"
   - "트렌드를 추출하여 표시한다"
   - "이상치를 탐지하여 표시한다"

4. **Visualization**: 시각화
   - "시계열 차트로 데이터를 표시한다"
   - "지도 위에 관측 지점별 데이터를 표시한다"

5. **Export**: 결과 출력
   - "분석 결과를 보고서 형태로 출력한다"
   - "차트를 이미지로 저장한다"

스프린트 배치:
- Sprint 1: Data Acquisition + 기본 Preprocessing
- Sprint 2: Analysis (핵심 통계)
- Sprint 3: Visualization
- Sprint 4: Export + 고급 Analysis
- Sprint 5: 통합 + Polish
